## Features
- [x]Collects data from Kraken Exchange, Bank of Canada, and European Central Bank
- [x]Stores data in DynamoDB
- [ ]Analyzes data
- [ ]Summarizes data
- [ ]User inputted orders
- [ ]System operated trading
- [ ]Machine Learning Predictions

## Technologies Used
- C# .Net Core
- WebAPI
- WebSocket
- DynamoDB
- JSON
- Threads
- ASync
